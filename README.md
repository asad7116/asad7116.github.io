# Portfolio homepage

This is a simple static homepage inspired by the image you uploaded. Files added:

- index.html
- css/styles.css
- js/script.js


Preview locally with a static server, e.g.:

```bash
cd "Asad Portfolio"
python3 -m http.server 8000
# then open http://localhost:8000
```

Place your uploaded hero image at `img/hero.jpg` (create the `img` folder inside the project root). The homepage now uses `img/hero.jpg` for the hero background. Add three project images as `img/project1.jpg`, `img/project2.jpg`, and `img/project3.jpg` so the project cards on the homepage display correctly.
